### Hướng dẫn cài đặt source code

#### Step 1: Clone source code về
```bash
    https://github.com/ngocquy-allgrow/htmltojson.git
```

### Step 2: Cài đặt node_modules
``` bash
    npm i
```
### Start serer
```bash
    node app.js
```
#### Api html to json html-to-json
```bash
    http://localhost:3000/html-to-json
```
